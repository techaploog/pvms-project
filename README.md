# PVMS Project

## Status
   MVP :  `completed`.

## Reqirements
   [NodeJs](https://nodejs.org/en/) : `^19.0.0`.

## Download Project
```
git clone https://github.com/techaploog/pvms-project.git
```
or download a [zip file](https://github.com/techaploog/pvms-project/archive/refs/heads/main.zip)

## Important !!
Make sure to set all of these ENVIRONEMENT variables.
```
#for client
PVMS_SRC_URL=""
PVMS_DESC_IP=""
PVMS_DESC_PORT=""
PVMS_LOCAL_PORT=""

#for server
PVMS_SERV_PORT=""
```

## Start Client Guide
```
cd pvms-project
npm install
npm run client
```

## Start Server 
```
npm run server
```
